# Система управления банковскими картами

## Описание
Данный проект позволяет управлять банковскими картами, пользователями и <br>
совершать операции перевода средств с карты на карту. В приложении <br>
установлено разграничение прав по ролям. Всего реализовано 2 роли: <br>
'Администратор' и 'Пользователь'. Для каждой операции пользователь должен<br> 
быть авторизован, в соответстви с его ролью.

## Распределение доступа по ролям
+ Администратор
   + Создает карту
   + Запрашивает постраничный список карт c фильтрацией
   + Запрашивает данные карты
   + Удаляет карту
   + Обновляет статус и/или срок действия карты
   + Создает данные аккаунта пользователя
   + Запрашивает постраничный список аккаунтов пользователей с фильтрацией
   + Удаляет аккаунт пользователя
   + Редактирует данные аккаунта пользователя
   + Запрашивает постраничный список запросов на блокировки карт с фильтрацией
   + Обновляет статус запроса на блокировку карты
+ Пользователь
   + Создает запрос на блокировку карты
   + Запрашивает баланс на карте
   + Запрашивает список всех своих карт
   + Делает переводы с карты на карту (только между своими картами)
+ Неаутентифицированный пользователь
   + Подтверждает в системе свою подлинность, чтобы получить права
  
## Особенности реализации
+ Пароли и номера карт хранятся в базе данных в зашифрованном виде
+ Номера карт при выдаче маскируются
+ Приложение поддерживает валидацию входящих запросов
+ Приложение отдает информативные сообщения об ошибках
+ Приложение логирует как ошибки, так и успешные операции

## Инструкция по сборке и запуску
1. Клонировать проект из репозитория https://github.com/DrEremin/bank-cards
2. Собрать проект в IDE
3. В файле аpplication.yml установить адреса, методы и заголовки для cors,<br> 
время жизни JWT и пароль для первого пользователя (его имя всегда root)
4. Проверить, что путь к собранному .jar-файлу в строке 'COPY' в Dockerfile<br> 
указан верно
5. Через терминал зайти в корневую директорию проекта
6. Создать в директории файл .env и прописать туда переменные окружения
   + POSTGRES_DB=имя базы данных
   + POSTGRES_USER=имя пользователя
   + POSTGRES_PASSWORD=пароль от бд
   + JWT_SECRET=секрет
   + TEXT_ENCRYPTOR_PASSWORD=пароль для кодировщика текста
   + FIRST_ADMIN_PASSWORD=пароль первого пользователя
7. Выполнить команду docker compose up -d
8. Приложение с бд должно запуститься
9. Доки по API доступны на http://localhost:8080/bank-cards/v3/api-docs.html
10. При первом запуске в базе данных автоматически будет создан первый <br> 
пользователь с ролью ADMIN и именем root. Его пароль можно менять в переменной <br>
FIRST_ADMIN_PASSWORD он будет обновлятся при каждом запуске
